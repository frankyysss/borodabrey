# BORODABREY — premium barbershop site

## Что внутри
- `index.html` — главная страница
- `styles.css` — стили
- `services.js` — все категории и услуги
- `script.js` — модалка, мобильное меню, кнопка прокрутки
- `assets/` — логотип и изображения

## Как открыть
Просто открой `index.html` в браузере.

## Что можно менять
- логотип: `assets/logo.svg`
- фото: файлы в `assets/`
- контакты и ссылки: в `index.html`
- услуги: `services.js`

## Что сделано
- тёмный premium стиль
- hero-секция под барбершоп
- модальное окно услуг
- адаптив под телефон
- footer с контактами и картами
- плавающая кнопка вверх/вниз
